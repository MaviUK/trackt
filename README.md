# TV Tracker

Track your favourite TV shows.

## Stack
- React + Vite
- Supabase Auth (Magic Link)
- Netlify Functions
- TheTVDB API

## Local setup
1. Copy `.env.example` to `.env`
2. Add your Supabase keys
3. Run `npm install`
4. Run `npm run dev`

## Features planned
- Search shows
- Airing today
- Upcoming episodes
- Watch tracking
- Push notifications
